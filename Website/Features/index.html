Non Desctructive Editing
Non destructive editing allows you to edit an image without overwriting the original. The edits are stored in a sperate file and a rendering is displayed on the screen. Export the rendering when you are happy with edits to a new image file.
RAW Files
Use PhoediX to work with RAW iamges.
8 / 16 Bit Image Processing
Choose the bit depth in settings to select working with 8 or 16 bits per channel processing. Allowing 16 bits per channel processing will allow for cleaner edits with fewer artficats.
Edit Pipeline
Use the Edit Pipeline to define the order of your edits. Move edits around easily, and copy/paste edit parameters between edits in the current project, or another open project.
Library
Use the built in Library to preview images in a single or in multiple directories. Quickly expand the images for closer inspection, open in a PhoediX project, and view details about the image.
Pixel Peeper
Pixel Peeper shows color information about the pixel at the location on an image from a right mouse click. Neighbor pixels of a selected size can be displayed as well. Right click on the pixel in the pixel peeper window to view color and location properties.
Available Edits
Edit Description
Adjust Brightness Adjust the brightness of the image with options to preserve details in shadows, highlights, or both.
Adjust Contrast Adjust the contrast using a linear or built in S curve function, to give the image a more vivid or flat look.
Adjust HSL Adjust the Hue, Saturtation, and Luminance (HSL) channels of the image.
Adjust LAB Adjust the Luminance, A, and B (LAB) channels of the image.
Adjust RGB Adjust the Red, Green, and Blue (RGB) channels of the image.
HSL Curves Adjust the Hue, Saturtation, and Luminance (HSL) channels of the image using custom defined curves.
LAB Curves Adjust the Luminance, A, and B (LAB) channels of the image using custom defined curves.
RGB Curves Adjust the Red, Green, and Blue (RGB) channels of the image using custom defined curves.
Convert to Greyscale Convert the image to greyscale, with three different options. Options include Average of RGB channels, human eyesight scale, or custom scale.
Channel Mixer Mix RGB master channels of the image with RGB subchannels for each channel.
Rotate Image Rotate the image 90, 180, or 270 degrees. Custom angles and cropping available as well, using nearest neighbor, bilinear, or bicubic interpolation.
Mirror Image Mirror the image vertically or horizontally.
Scale Image Scale and resize the image to any size (with or without preserving the aspect ratio).
Crop Image Crop the image (with or without preserving the aspect ratio).
Blur Image Use Fast Blur to blur the image horizontally, vertically, or in both directions. Specify the blur size and number of iterations.
